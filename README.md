This repository contains the source codes and scripts used to generate the results presented in the paper "Salt cavern simulations with a stabilized mixed finite element formulation for low-order tetrahedral elements".

## Requirements

This project requires **FEniCSx 0.9.0**, which cannot be installed via pip.

Install FEniCSx using one of the following methods:

- Conda (recommended)
- Docker
- System packages

After installing FEniCSx 0.9.0, install the Python dependencies:

```bash
pip install -r requirements.txt# paper-mixed-salt
